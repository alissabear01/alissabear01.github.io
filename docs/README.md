## About

This is my creative portfolio, built to showcase three sides of my work:

- **Art & Illustration** — paintings, illustrations, and character work
- **Game Development** — games built in Unity and Godot
- **Graphic Design** — branding, posters, and visual design projects

---

## Built With

- Pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies
- Hosted on [GitHub Pages](https://pages.github.com/)
- Fonts via Google Fonts (Cormorant Garamond + Jost)
